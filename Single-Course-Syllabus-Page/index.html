Welcome to the Course!

This course will teach you the fundamentals of HTML (Hypertext Markup Language). HTML is the standard markup language used for creating web pages and web applications.

Course Overview

In this Course, you will learn:

  • The basic structure of an HTML document
  • Common HTML tags and their uses
  • How to create links,images and tables
  • HTML forms and input elements
  • HTML5 semantic elements
